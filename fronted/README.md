## Project Overview
The **Speech-to-Text App** is a web application that converts spoken words into text in real-time using the **Web Speech API**.  
It is built with **React**, **Vite**, and **Tailwind CSS**, providing a clean and responsive interface for users to easily dictate text through their microphone.

---

## Features
- Real-time speech recognition
- **Start**, **Stop**, and **Clear** buttons for controlling speech input
- Responsive design for desktop and mobile
- Styled using Tailwind CSS for a modern UI
- Easy to run locally with minimal setup
